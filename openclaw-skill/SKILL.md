# glitch-stats Skill

Display agent gamification stats (RPG-style progression system).

## Commands

- `/rpg-stats` — Show current character status dashboard
- `/rpg-quests` — View quest history
- `/rpg-achievements` — Show unlocked and available achievements

## Usage

```
/rpg-stats
```

Displays level, XP, tasks, streak, skills, and achievements in terminal-friendly ASCII format.

## Implementation

This skill executes the glitch-stats Node.js commands from the workspace and displays output in-chat.
